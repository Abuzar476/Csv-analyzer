# Csv-Data-analyzer
This program loads ONLY csv files, and will give error if its not a csv file.
It shows the shape, column types, missing values and stats.
It will also generate a histograms and a correlation heatmap.
The titanic file can be used to test the program, you can also add any other csv file to use this.
Run the main.py file and ust enter the file path and it will give you the details.
